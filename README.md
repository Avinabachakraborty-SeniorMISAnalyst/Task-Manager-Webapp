# Task Manager Web App

A task management system built on Google Apps Script + Google Sheets.

## Features
- Director and Employee web app portals
- User ID / Password login system
- Task assignment with email notifications
- Performance Dashboard with scoring
- Mobile-friendly design

## Files
- `Code.gs` — Backend (Google Apps Script)
- `director.html` — Director web app
- `employee.html` — Employee web app

## Setup
1. Create a Google Sheets spreadsheet
2. Open Apps Script (Extensions → Apps Script)
3. Paste Code.gs, director.html, employee.html
4. Update SPREADSHEET_ID in Code.gs
5. Deploy as Web App
